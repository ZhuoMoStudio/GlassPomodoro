# GlassPomodoro

🍅 横屏玻璃风格番茄时钟 — 内置白噪音 + 动态水波 + 壁纸系统

## v1.0.5 特性

- 🕐 **大字时钟** — 5 种字体切换，自定义颜色
- 🌊 **动态水波纹** — 音频驱动声波涟漪
- 🖼️ **多源壁纸** — Bing每日壁纸 / 本地相册
- 🎭 **4种遮罩样式** — 径向渐变 / 动态光晕 / 毛玻璃 / 科技网格（透明度 0~1 可调）
- 🌓 **主题模式** — 深色 / 浅色 / 跟随系统
- 🍅 **番茄专注** — 25分钟专注计时
- 🔊 **内置白噪音** — 6种 CC0 高品质音频（雨/海浪/篝火/森林/溪流/白噪音）
- 🌐 **中英文双语**

## 技术栈

- Kotlin + Jetpack Compose
- Material 3
- Room（专注统计）
- DataStore（设置持久化）
- Coil（壁纸加载）
- Android Visualizer API（音频可视化）
